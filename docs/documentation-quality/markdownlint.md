# Markdownlint

## Overview

Markdownlint helps maintain consistent Markdown formatting across the documentation.

## Benefits

Markdownlint helps:

- Maintain consistent formatting.
- Reduce formatting errors.
- Improve readability.
- Simplify pull request reviews.

## Current configuration

The handbook uses **markdownlint-cli2** with a custom `.markdownlint.json` configuration.
