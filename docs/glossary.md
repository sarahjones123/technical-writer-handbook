# Glossary

## Purpose

This glossary defines commonly used documentation and technical writing terms.

| Term | Definition |
|------|------------|
| Docs-as-Code | Managing documentation using software development practices. |
| Markdown | A lightweight markup language used for writing documentation. |
| MkDocs | A static site generator for project documentation. |
| Markdownlint | A tool for validating Markdown formatting. |
| Vale | A prose linter for checking writing quality. |
| Style guide | A document that defines writing standards and conventions. |
