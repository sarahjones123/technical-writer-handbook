# Terminology

## Overview

Consistent terminology improves clarity and helps readers understand the documentation. Use approved terms consistently throughout the documentation.

## Guidelines

Follow these guidelines when using terminology:

- Use consistent terminology.
- Use product terminology consistently.
- Define unfamiliar acronyms on first use.
- Avoid using multiple terms for the same concept.
- Use industry-standard terminology whenever possible.

## Best practices

### Be consistent

**Instead of**

```text
Customer
Client
Account
```

Use a single approved term throughout the documentation.

### Define acronyms

```text
Local Distribution Company (LDC)
```

Use **LDC** in subsequent references.

## Checklist

Before publishing documentation, verify that:

- Terminology is consistent.
- Acronyms are defined on first use.
- Product terminology matches the application.
- Synonyms are avoided.
