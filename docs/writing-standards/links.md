# Links

## Overview

Links help readers navigate related information. Use descriptive link text that clearly communicates the destination.

## Guidelines

Follow these guidelines when adding links:

- Use meaningful link text.
- Avoid "click here" or "read more."
- Link only when it adds value.
- Verify that links are valid.

## Best practices

### Use descriptive links

**Instead of**

```text
Click here.
```

**Write**

```text
For more information, see Configure billing rules.
```

### Avoid unnecessary links

Only add links that help the reader complete a task or understand a concept.

## Checklist

Before publishing documentation, verify that:

- Link text is descriptive.
- Links are relevant.
- Links are valid.
- Broken links are removed.
