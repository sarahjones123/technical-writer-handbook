# Images

## Overview

Images should support the content by helping readers understand a concept or complete a task. Do not use images as a substitute for written instructions.

## Guidelines

Follow these guidelines when using images:

- Use images only when they add value.
- Keep screenshots clear and readable.
- Crop images to highlight relevant content.
- Keep annotations simple.
- Update images when the UI changes.

## Best practices

### Use screenshots when necessary

Use screenshots to highlight complex user interface elements.

### Keep images focused

Show only the relevant portion of the screen.

### Use annotations sparingly

Use arrows or highlights only when they improve clarity.

## Checklist

Before publishing documentation, verify that:

- Images are clear.
- Images support the content.
- Images are current.
- Images are appropriately cropped.
- Annotations are easy to understand.
