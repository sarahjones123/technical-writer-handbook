# Writing standards

## Overview

Writing standards ensure that documentation remains clear, consistent, and easy to understand regardless of who writes it.

This handbook adopts the **Google Developer Documentation Style Guide** as the primary writing standard.

Company-specific writing standards will supplement Google's guidance when necessary.

## Principles

Focus on:

- Voice and tone
- Formatting
- Consistency
- Readability
- User-focused writing
